# 🏆 FIFA 2026 World Cup Bracket Predictor

An interactive, full-stack tournament simulation web application built with **Next.js** and **Antigravity**, featuring an automated AI infographic generation pipeline using **Gemini 3.5 Flash**. Users can build, track, and fully visualize their prediction brackets from the initial group stages all the way to the Grand Final.

🌐 **Live Deployment:** [fifa-2026-wc-simulator.vercel.app](https://fifa-2026-wc-simulator.vercel.app/)

---

## 🚀 Key Features

* **Complete Tournament Mapping:** Interactive nodes handling the progression flow from the group stages down to the ultimate champion.
* **AI Infographic Generation:** Custom backend integration that structures user prediction brackets into high-quality, share-ready PDFs and aesthetic image cards optimized for social reach.
* **Responsive Architecture:** Clean, high-performance user interface styled for fluid interaction across mobile layouts and desktop dashboards.

## 🛠️ Tech Stack

* **Framework:** Next.js (App Router)
* **AI Pipeline:** Gemini 3.5 Flash via Antigravity 
* **Deployment:** Vercel

## 💻 Local Development & Usage Guide

Follow these steps to download, install, and run the project locally on your machine.

### 1. Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed (v18.x or higher recommended).

### 2. Download and Setup
Open your terminal or command prompt and run the following commands:

```bash
# Clone the repository
git clone [https://github.com/Soumyaexe2004/fifa-2026-bracket-builder.git](https://github.com/Soumyaexe2004/fifa-2026-bracket-builder.git)

# Navigate into the project directory
cd fifa-2026-bracket-builder

# Install dependencies
npm install
# or yarn install / pnpm install / bun install

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `app/page.tsx`. The page auto-updates as you edit the file.

This project uses [`next/font`](https://nextjs.org/docs/app/building-your-application/optimizing/fonts) to automatically optimize and load [Geist](https://vercel.com/font), a new font family for Vercel.
